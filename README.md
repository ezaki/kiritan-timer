# きりぴょんタイマー
![zzm_kiritan_tou](https://cloud.githubusercontent.com/assets/13980441/15290186/bc5e11a6-1bb1-11e6-96ef-b85669099db9.png)

##使い方
[http://keika299.github.io/kiritan-timer/](http://keika299.github.io/kiritan-timer/)に設置しています。
